# Frontend - URL to TXT Converter

Minimal React frontend for the URL to TXT Converter service.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables:
   Copy `.env.example` to `.env` and adjust `VITE_API_URL` if necessary.

3. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment on Vercel

1. Push your code to a GitHub repository (separately for frontend and backend if desired, or use monorepo settings).
2. Connect your repository to Vercel.
3. Configure the environment variables on the Vercel dashboard.
